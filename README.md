# multilingual-mental-health-llm-evaluation

Code, prompts, scripts, notebooks, and supporting materials for an MSc dissertation on multilingual evaluation of LLM responses to mental-health problem statements in English and Hindi.

## Repository structure
- `scripts/` – Python scripts used for generation and preprocessing
- `notebooks/` – model notebooks for Zero-Shot and CoT runs
- `data/` – dataset-related files or placeholders
- `outputs/` – generated outputs
- `figures/` – images/figures used in the dissertation
- `docs/` – supporting documentation

## Models used
- GPT-2
- GPT-2 Medium
- GPT-Neo
- GPT-3.5
- LLaMA-2 7B
- Nemotron

## Prompting strategies
- Zero-Shot
- Chain-of-Thought (CoT)

## Notes
This repository contains code and materials used to generate model responses for dissertation experiments in English and Hindi.
