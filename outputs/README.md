# Model Outputs (English & Hindi)

This repository contains the generated responses from multiple Large Language Models evaluated in this study.

## Models Included
- GPT-3.5
- LLaMA
- Nemotron
- Distil_vs_mBERT_vs_XLMR

## Description
Each folder contains model outputs for the same set of prompts in:
- English (EN)
- Hindi (HI)

Two prompting strategies were used:
- Zero-shot prompting
- Chain-of-Thought (CoT) prompting

## Purpose
These outputs were used for:
- Human evaluation (Emotional Appropriateness, Safety, Helpfulness)
- Automated evaluation (semantic similarity, sentiment alignment, etc.)
